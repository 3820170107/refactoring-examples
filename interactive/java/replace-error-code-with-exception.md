###

```
class Account {
  // ...
  private int balance;

  public int withdraw(int amount) throws BalanceException {
    if (amount > balance) {
      throw new BalanceException();
    }
    balance -= amount;
  }
}
class BalanceException extends Exception {}

// Somewhere in client code.
try {
  account.withdraw(amount);
  doTheUsualThing();
} catch (BalanceException e) {
  handleOverdrawn();
}
```
